# Guide d'utilisation de l'application

## Accès

Application en ligne : [TON_LIEN_HUGGING_FACE]

Aucune installation nécessaire. Fonctionne directement dans le navigateur.

## Présentation générale

L'application propose cinq modules d'analyse accessibles via des onglets distincts.

## Module 1 : Analyse par pays

Permet de visualiser l'évolution historique et les projections pour un pays spécifique.

Procédure :
1. Sélectionner un pays dans la liste déroulante
2. Cliquer sur le bouton d'analyse
3. Consulter les deux graphiques générés

Interprétation :
- Graphique gauche : trajectoire historique 1991-2024
- Graphique droite : projection tendancielle 2025-2030
- Statistiques affichées : niveau actuel, moyenne historique, tendance

## Module 2 : Comparaison bilatérale

Compare l'évolution de deux pays sur la période complète.

Procédure :
1. Choisir deux pays différents
2. Lancer la comparaison
3. Analyser le graphique superposé

Utilité : identifier les convergences ou divergences entre trajectoires nationales.

## Module 3 : Classification automatique

Identifie des groupes de pays aux profils économiques similaires.

Procédure :
1. Cliquer sur le bouton d'analyse
2. Observer le graphique de dispersion

Lecture du graphique :
- Position horizontale : niveau de dette publique
- Position verticale : taux de croissance économique
- Couleurs : appartenance aux groupes identifiés
- Proximité spatiale = similarité des profils

## Module 4 : Performance du modèle

Évalue la qualité des prédictions et l'importance des variables.

Procédure :
1. Afficher les métriques
2. Consulter les deux visualisations

Premier graphique : importance relative des variables explicatives
Second graphique : qualité de l'ajustement (prédictions vs observations réelles)

## Module 5 : Export des résultats

Génère un fichier Excel téléchargeable avec l'ensemble des données et projections pour un pays.

Procédure :
1. Sélectionner le pays d'intérêt
2. Générer le rapport
3. Télécharger le fichier produit

Contenu du fichier :
- Premier onglet : série historique complète
- Second onglet : projections 2025-2030

## Recommandations d'usage

Pour une analyse approfondie :
1. Débuter par l'analyse individuelle d'un pays
2. Comparer avec des pays de référence
3. Identifier le groupe d'appartenance via le clustering
4. Vérifier la robustesse via les métriques de performance

Pour une présentation :
- Les graphiques peuvent être téléchargés via le menu contextuel
- Format PNG haute résolution disponible

## Interprétation des projections

Les projections sont des scénarios tendanciels basés sur la continuité des relations passées. Elles ne constituent pas des prévisions officielles.

Une projection en hausse indique que, toutes choses égales par ailleurs, la dette publique devrait augmenter si les tendances récentes se maintiennent.

## Précautions méthodologiques

- Les résultats sont plus fiables pour les pays avec séries complètes
- Les chocs imprévus ne sont pas modélisés
- Les projections à long terme (2028-2030) sont plus incertaines
- Toujours contextualiser les résultats avec l'actualité économique
